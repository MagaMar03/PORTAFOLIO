Lenguaje Marcado de Hipertexto HTML y Lenguaje Marcado Extensible XML

¿Qué aprendí?

Aprendí que construir una página web moderna implica dominar diversos lenguajes y estándares que conforman la Open Web Platform, siendo HTML y XML la base fundamental para estructurar el contenido. HTML (Lenguaje de Marcado de Hipertexto) es esencial para definir la estructura visual y semántica de una página web, utilizando etiquetas principales como "header", "nav", "section", "article" y "footer" introducidas con HTML5. Por otro lado, XML (Lenguaje de Marcado Extensible) se usa más para transportar y almacenar datos de forma estructurada, lo que permite separar el contenido de la presentación. También comprendí cómo ambos lenguajes pueden integrarse en el desarrollo web para mejorar la organización y accesibilidad del contenido.

Asimismo, aprendí sobre la estructura jerárquica del Árbol DOM (Document Object Model), que permite a los navegadores interpretar y renderizar una página como un árbol de nodos, donde cada nodo representa un elemento HTML. Entendí que el ciclo de vida de una página web incluye fases como la carga, el procesamiento del DOM, la ejecución de scripts y el renderizado. Además, conocí la importancia del SEO (Search Engine Optimization), que permite que una página web sea más visible en los motores de búsqueda, utilizando prácticas como la correcta estructuración del HTML, el uso de etiquetas semánticas y atributos como alt, title, y metadatos. También comprendí que es fundamental garantizar la compatibilidad entre navegadores, ya que cada uno puede interpretar ciertos elementos de forma diferente, lo cual se resuelve mediante el uso de estándares y pruebas en múltiples entornos.

¿Cómo aprendí?

Aprendí este tema a través de una combinación de teoría y práctica. En primer lugar, estudié los conceptos fundamentales de HTML y XML mediante recursos como manuales, clases virtuales, videos explicativos y páginas de documentación oficial como MDN Web Docs. Estos recursos me permitieron entender la estructura básica de un documento HTML y cómo se diferencian las etiquetas de bloque, en línea e inline-block. También utilicé ejemplos prácticos para identificar cómo se define una estructura XML válida y cómo interactúa con otras tecnologías web. De esta forma, pude visualizar cómo se organiza internamente una página web y cómo se comunica con el navegador a través del Árbol DOM.

Además, desarrollé varias actividades prácticas como crear páginas HTML, experimentar con distintos elementos y validar la estructura mediante herramientas en línea. Posteriormente, incorporé hojas de estilo CSS3 para mejorar la presentación de la página. Esto incluyó el uso de sintaxis básica y avanzada de CSS, aplicación de unidades de medida absolutas (como px, cm) y relativas (como em, %), y la implementación de técnicas modernas como Flexbox y Grid Layout. También exploré cómo aplicar tipos de position (static, relative, absolute, fixed, sticky) y cómo diseñar sitios web con diseño fluido y responsivo, ajustando el contenido a diferentes tamaños de pantalla. Mediante la experimentación, el análisis de errores y la retroalimentación, logré consolidar mis conocimientos y desarrollar competencias prácticas en diseño web.
