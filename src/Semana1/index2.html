Reflexión: Fundamentos de las Tecnologías Web

¿Qué aprendí?

Aprendí que las tecnologías web son el corazón del Internet moderno. Ahora comprendo que existen diferencias clave entre un sitio web, una página web, una aplicación web y un sistema web, cada uno con sus funcionalidades específicas. Conocí que el HTML estructura el contenido, el CSS le da estilo y el JavaScript lo hace interactivo. Además, me familiaricé con el funcionamiento de la web a través de tecnologías como el DNS, el protocolo TCP/IP y HTTP, que son fundamentales para que una página web pueda ser accedida desde cualquier parte del mundo.

También aprendí sobre los roles importantes en el desarrollo web. Comprendí qué hace un desarrollador frontend, centrado en el diseño visual y la experiencia del usuario; un backend, que trabaja con la lógica y la base de datos; y el fullstack, que maneja ambos aspectos. Esta visión me permite entender cómo se construyen las soluciones web de manera colaborativa. Además, conocí herramientas como Visual Studio Code, que se ha vuelto esencial para escribir y gestionar el código de manera organizada, con muchas funciones útiles para mejorar la productividad.

¿Cómo aprendí?

Aprendí a través de la práctica y la exploración guiada. Utilicé recursos como tutoriales interactivos, documentación oficial, y videos educativos que explicaban cada parte paso a paso. Al comenzar creando sitios simples con HTML y luego agregando estilos con CSS, pude experimentar directamente cómo se transformaban visualmente. Fue clave equivocarme y corregir errores, porque me obligó a investigar más y a entender por qué sucedían ciertos comportamientos.

Visual Studio Code fue una herramienta esencial en este proceso. Me permitió organizar mi trabajo, usar extensiones como Live Server para ver los cambios en tiempo real, y tener resaltado de sintaxis que facilitaba mucho la lectura del código. El ambiente de trabajo fue amigable y eso hizo que aprender programación web fuera menos intimidante. Con cada proyecto, fui ganando confianza en mi capacidad para aplicar lo aprendido y comprender mejor cómo se construyen las soluciones tecnológicas que usamos a diario.
