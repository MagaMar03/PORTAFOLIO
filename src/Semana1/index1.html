Semana 1

Fundamentos de las Tecnologías Web

Las tecnologías web permiten la comunicación e interacción entre usuarios y sistemas a través de Internet, utilizando navegadores y servidores conectados mediante protocolos estándar.

Soluciones Web

  • Sistema web: Conjunto de componentes interconectados que gestionan procesos en línea.
  • Aplicación web: Programa accesible mediante un navegador, que realiza funciones específicas.
  • Sitio web: Colección de páginas relacionadas bajo un mismo dominio.
  • Página web: Documento individual visualizado mediante un navegador web.

Tecnologías Web

  • Básicas: HTML, CSS y JavaScript para estructurar, diseñar e interactuar.
  • Lenguajes: PHP, Python, Java, Go, entre otros.
  • Gráficos: SVG, WebGL, Canvas para renderizar imágenes y gráficos.

Funcionamiento de la Web

  • DNS: Sistema que traduce nombres de dominio en direcciones IP.
  • TCP/IP: Conjunto de protocolos que permite la transmisión de datos en redes.
  • HTTP: Protocolo de comunicación para acceder a páginas web.

Roles en el Desarrollo Web

  • Frontend Developer: Diseña la interfaz y experiencia del usuario.
  • Backend Developer: Gestiona la lógica, bases de datos y servidores.
  • Fullstack Developer: Combina habilidades frontend y backend para construir aplicaciones completas.
