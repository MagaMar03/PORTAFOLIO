Front-End con Javascript y Typescript-Manejar el DOM con Javascript y Canvas

¿Qué aprendí?

Aprendí que JavaScript es un lenguaje esencial para dotar de interactividad a las páginas web. Su ejecución se realiza a través de motores como V8 (usado por Google Chrome), lo que permite interpretar y ejecutar código en el navegador de forma rápida. Comprendí la importancia de declarar variables utilizando var, let y const, y cómo emplear operadores para realizar cálculos, comparaciones y asignaciones. Además, identifiqué los distintos tipos de datos como números, cadenas de texto (strings), objetos, booleanos y otros tipos como undefined, null, symbol y bigint. También fortalecí mis conocimientos sobre estructuras de control como if, switch y bucles como for, while, y forEach, que son fundamentales para el flujo lógico de un programa.

También aprendí sobre los distintos tipos de funciones en JavaScript, incluyendo funciones declaradas, funciones anónimas, funciones flecha (=>), funciones como objetos (Function), y closures, que permiten mantener el estado interno entre ejecuciones. Descubrí cómo usar funciones auto-invocadas (IIFE) para ejecutar lógica sin contaminar el ámbito global. Con respecto a TypeScript, aprendí que es un superset de JavaScript que añade tipado estático, lo que ayuda a detectar errores en tiempo de desarrollo y escribir código más robusto y mantenible. TypeScript me permitió definir tipos de datos explícitos, interfaces, clases y usar características modernas de JavaScript de forma más estructurada, lo cual mejora la legibilidad del código y facilita el trabajo en proyectos grandes.

¿Cómo aprendí?

Aprendí mediante una combinación de teoría, ejercicios prácticos y desarrollo de proyectos sencillos como parte del Laboratorio 04, donde desarrollé una animación con JavaScript. Comencé explorando cómo manipular el DOM (Document Object Model) para acceder y modificar elementos HTML utilizando métodos como getElementById, querySelector, y propiedades como innerHTML o style. Esto me permitió cambiar dinámicamente los contenidos y estilos de una página. A través de ejemplos interactivos, comprendí cómo JavaScript puede escuchar eventos (onclick, onchange, keydown) y ejecutar funciones en respuesta a la interacción del usuario, haciendo la página más dinámica y reactiva.

Además, aprendí a trabajar con el elemento de HTML5 para desarrollar animaciones y gráficos básicos. Utilizando el contexto 2d, experimenté con métodos como fillRect, arc, lineTo, y stroke, lo cual me permitió dibujar figuras, mover objetos y crear efectos visuales personalizados. Durante el laboratorio, desarrollé una animación donde un objeto se desplazaba por la pantalla, aplicando conceptos como coordenadas, intervalos de tiempo con setInterval y control de colisiones. Gracias a la integración de JavaScript con Canvas, logré comprender cómo se pueden combinar gráficos y lógica de programación para crear experiencias interactivas y visualmente atractivas. Todo este aprendizaje fue reforzado con práctica constante, revisión de errores y el uso de herramientas como la consola del navegador para depuración.
