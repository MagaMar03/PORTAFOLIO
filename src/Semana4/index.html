Laboratorio 04: Desarrollo de una animación con JavaScript

¿Qué es JavaScript y TypeScript?

JavaScript es un lenguaje de programación que permite agregar interactividad a páginas web. TypeScript es una versión mejorada de JavaScript con tipado estático.

Motores de JavaScript

Los navegadores usan motores para interpretar JS, como V8 (Chrome) y SpiderMonkey (Firefox).

Variables y Operadores

// JavaScript
let nombre = "Ana";
const edad = 25;
let activo = true;

// TypeScript
let nombre: string = "Ana";
let edad: number = 25;

Tipos de Datos

  • Número: let x: number = 10;
  • String: let saludo: string = "Hola";
  • Booleano: let activo: boolean = true;
  • Objeto: let persona = { nombre: "Ana", edad: 22 };

Estructuras de Control y Bucles

if (edad > 18) {
  console.log("Mayor de edad");
} else {
  console.log("Menor de edad");
}

for (let i = 0; i < 3; i++) {
  console.log(i);
}

Funciones

  • Declarativa: function saludar(nombre) { return "Hola " + nombre; }
  • Flecha: const suma = (a, b) => a + b;
  • Auto invocada: (function(){ console.log("Hola") })();
  • Closure: 
    function contador() {
  let c = 0;
  return function() { c++; return c; }
}
let incrementar = contador();
