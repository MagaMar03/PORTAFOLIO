Laboratorio 05A: Desarrollo Front-End con Framework JS

1. Estructura y operación de Framework - Client Side Rendering

Los frameworks como React, Vue o Angular utilizan Client Side Rendering (CSR), lo que significa que el navegador se encarga de renderizar el contenido a partir de JavaScript sin recargar la página completa. El contenido se genera dinámicamente en el cliente.

2. Gestión de dependencias y herramientas de creación

Para crear proyectos con frameworks modernos, se utilizan herramientas como:

  • npm o yarn: gestores de paquetes.
  • Vite, Create React App, Vue CLI: inicializadores de proyectos.
  • Webpack: herramienta de empaquetado.
npx create-react-app mi-app
cd mi-app
npm start

4. Estilos en Frameworks JS

  • Inline Styles: <div style={{ color: 'red' }}>
  • Style Sheets: Importar CSS normal con import './App.css'
  • CSS Modules: Archivos como styles.module.css con nombres locales.
  • Styled Components: Estilo escrito en JS con librerías como styled-components.
  • Framework CSS: Como Tailwind CSS.
