FrontEnd con Framework JS

¿Qué aprendí?

Aprendí que desarrollar aplicaciones Front-End con un Framework JavaScript como React permite crear interfaces dinámicas y escalables de forma mucho más eficiente que utilizando solo HTML, CSS y JavaScript puro. Uno de los conceptos más importantes que adquirí fue el de Client Side Rendering (CSR), que permite que el contenido de la página se genere directamente en el navegador del usuario, lo que mejora la experiencia de navegación al hacer las aplicaciones más rápidas y reactivas. También comprendí la importancia de las herramientas de gestión de dependencias, como npm o yarn, y cómo utilizar comandos como npx create-react-app para inicializar rápidamente un proyecto con todas las configuraciones necesarias.

Otro aspecto clave que aprendí fue el desarrollo basado en componentes, donde cada parte de la interfaz (como botones, formularios o tarjetas) se desarrolla como una unidad independiente y reutilizable. Este enfoque facilita la organización del código y mejora la mantenibilidad del proyecto. Usando JSX, una extensión de JavaScript que permite escribir código HTML dentro del archivo JavaScript, pude crear componentes de forma declarativa. También aprendí a pasar información entre componentes usando props (propiedades), que permiten enviar datos desde un componente padre a uno hijo, y a utilizar children para renderizar contenido anidado de manera flexible. En el Laboratorio 05A, practiqué cómo enviar estos datos y mostrarlos correctamente en un componente hijo, lo que me permitió consolidar este conocimiento con un ejemplo práctico.

¿Cómo aprendí?

Aprendí mediante una combinación de teoría, tutoriales interactivos y práctica constante en el entorno de desarrollo de React. Al utilizar herramientas modernas como Visual Studio Code, y comandos de terminal para gestionar dependencias y levantar servidores locales (npm start), comprendí cómo se estructura y opera un proyecto dentro de un framework. En el proceso, también utilicé documentación oficial y recursos en línea para familiarizarme con conceptos fundamentales como el ciclo de vida de los componentes, el estado local (useState), y el renderizado condicional, lo cual me dio una base sólida para construir aplicaciones más completas.

En cuanto a los estilos, aprendí varias formas de aplicar diseño visual dentro del framework: usando estilos en línea, hojas de estilo CSS tradicionales, CSS Modules (para encapsular estilos por componente) y librerías como Styled Components, que permiten escribir estilos directamente dentro del archivo del componente usando JavaScript. Además, también exploré cómo integrar frameworks CSS como Bootstrap o Tailwind con React para agilizar el diseño visual. Esta variedad de enfoques me dio la flexibilidad de elegir la técnica más adecuada según el contexto del proyecto. Finalmente, el desarrollo del laboratorio me ayudó a practicar cómo componer componentes padres e hijos, enviar datos con props, y renderizar contenido dinámico usando children, todo mientras aplicaba estilos y buenas prácticas.
